## 종합프로젝트 4조 소스코드입니다.

주제: 독거노인 고독사 방지 솔루션

## Folder Structure

- `App`: the folder to maintain Android sources
- `Arduino`: the folder to maintain Arduino sources
- `Server`: the folder to maintain Spring Boot sources

기술스택

  Server: SpringBoot, JAVA, JPA
  DB: H2_DB
  App: Kotlin
  Arduino: Arduino

팀장: 장정우
팀원: 권민혁, 이호왕, 지현종
